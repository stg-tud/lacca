# Project Title

lacca


## Installation

Clone the ucan-scala project

```bash
  git clone https://github.com/stg-tud/daimpl-lofi-auth.git
```

Create a terminal window

```bash
  cd ucan-scala
  sbt +publishLocal
```

Clone the lacca project

```bash
  git clone https://github.com/stg-tud/lacca.git
```

Install lacca with npm

```bash
  cd lacca
  npm install
```

Create a terminal window

```bash
  sbt
  sbt:kanban> ~fastLinkJS
```

Create another terminal window

```bash
  npm run dev
```
